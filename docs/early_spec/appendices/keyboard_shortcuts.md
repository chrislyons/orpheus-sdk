# Keyboard Shortcuts Reference

## Core Operations

### Global Operations
- Ctrl+N: New session
- Ctrl+O: Open session
- Ctrl+S: Save session
- Ctrl+Shift+S: Save as...
- Ctrl+W: Close session
- Ctrl+Q: Quit application
- F1: Help/Documentation

### Transport Controls
- Space: Play/Stop
- Return: Reset to start
- Shift+Space: Emergency stop
- [/]: Scrub position
- Ctrl+T: Tap tempo
- Shift+T: Set tempo
- Alt+T: Toggle tempo sync

### Edit Operations
- Ctrl+Z: Undo
- Ctrl+Shift+Z: Redo
- Ctrl+X: Cut
- Ctrl+C: Copy
- Ctrl+V: Paste
- Delete: Clear cell

### Grid Navigation
- Tab: Next cell
- Shift+Tab: Previous cell
- Arrow keys: Navigate grid
- Ctrl+Arrow: Jump to edge
- Home/End: First/Last cell
- PgUp/PgDn: Previous/Next tab
- Ctrl+1-8: Select tab
- Ctrl+Tab: Next page (dual mode)

### Clip Operations
- Enter: Edit selected clip
- Ctrl+E: Show/hide editor
- Ctrl+B: Set IN point
- Ctrl+N: Set OUT point
- Ctrl+L: Toggle loop
- Ctrl+K: Add cue marker
- Ctrl+W: Add warp marker
- Ctrl+G: Assign to group

### Group Operations
- Alt+A-D: Select group
- Shift+A-D: Toggle group
- Ctrl+Alt+A-D: Solo group
- Ctrl+Shift+A-D: Mute group

## Media Management

### Project Media
- Ctrl+Shift+M: Open media folder
- Ctrl+Shift+F: Find missing files
- Ctrl+Shift+R: Refresh media
- Ctrl+Alt+F: Show file info
- Ctrl+Alt+M: Media browser
- Ctrl+Alt+O: Organize files
- Ctrl+Alt+V: Validate files
- F2: Rename selected

### Import Operations
- Ctrl+I: Import files
- Ctrl+Shift+I: Import folder
- Ctrl+Alt+I: Import and analyze
- Ctrl+D: Show duplicates
- Ctrl+Alt+B: Batch import
- Ctrl+Alt+T: Import settings
- Ctrl+Alt+P: Import presets
- F3: Quick import

## Logging System

### Log Operations
- Ctrl+Shift+L: Show logs
- Ctrl+Alt+L: Export logs
- Ctrl+Alt+E: Export settings
- F4: Toggle log window

### Cloud Integration
- Ctrl+Alt+S: Sync now
- Ctrl+Shift+S: Sync settings
- Ctrl+Alt+C: Cloud status
- F5: Refresh sync status

### Export Shortcuts
- Ctrl+E: Quick export (default format)
- Ctrl+Shift+E: Export as...
  * 1: CSV format
  * 2: JSON format
  * 3: XML format
- Ctrl+Alt+A: Export all logs

## Platform-Specific

### Windows
- Alt+Enter: Full screen
- Win+Shift+S: Screenshot
- Alt+Space: System menu
- F11: Toggle window mode

### macOS
- Cmd+M: Minimize
- Cmd+`: Cycle windows
- Cmd+Shift+F: Full screen
- Cmd+Ctrl+F: Picture in Picture

### iOS
- Two-finger tap: Context menu
- Three-finger swipe: Undo/Redo
- Four-finger pinch: Home
- Edge swipe: Back

### Format Management
- Ctrl+Alt+R: Resource monitor
- Ctrl+Alt+E: Emergency protocol status
- F6: Format conversion panel
- F7: Resource usage panel
- F8: Emergency protocol panel
