# Keyboard Command Reference

## Transport Controls
- **Playback**
  * SPACE: Play/Stop
  * SHIFT+SPACE: Emergency Stop
  * ←/→: Nudge Position
  * SHIFT+←/→: Fine Position Control

## Page Controls
- **View Mode**
  * F3: Toggle Single/Dual Page
  * F4: Swap Page Contents
  * ALT+1/2: Focus Page 1/2
  * CTRL+E: Export Active Page
  * CTRL+SHIFT+I: Import Page

## Edit Controls
- **Clip Editing**
  * E: Open/Close Edit Panel
  * I: Set IN Point
  * O: Set OUT Point
  * L: Toggle Loop
  * SHIFT+I/O: Fine Adjust IN/OUT
  * DELETE: Remove Clip
  * F1: Clear custom fades (revert to global)
  * Shift+F: Toggle power compensation
  * Alt+Left/Right: Adjust IN fade
  * Alt+Up/Down: Adjust OUT fade
  * Ctrl+Alt+Left/Right: Change curve type

## System Controls
- **Application**
  * CTRL+N: New Session
  * CTRL+O: Open Session
  * CTRL+S: Save Session
  * CTRL+Z: Undo
  * CTRL+SHIFT+Z: Redo

## Grid Navigation
- **Grid Navigation**
  * CTRL+TAB: Next Tab in Active Pane
  * CTRL+SHIFT+TAB: Previous Tab in Active Pane
  * 1-8: Direct Tab Selection
  * A/B/C/D: Group Selection
