Job Command Description
Build & Compile
Configure CMake cmake -S . -B build -DCMAKE_BUILD_TYPE=Debug Configure CMake build system (Debug mode)
Configure CMake (Release) cmake -S . -B build -DCMAKE_BUILD_TYPE=Release Configure CMake build system (Release mode, optimized)
Build SDK cmake --build build Build all SDK targets
Build SDK (parallel) cmake --build build -j$(sysctl -n hw.ncpu) Build using all CPU cores for faster compilation
Build Clip Composer cmake --build build --target orpheus_clip_composer_app Build only the Clip Composer application (Debug)
Build Clip Composer (Release) cmake -S . -B build-release -DCMAKE_BUILD_TYPE=Release && cmake --build build-release --target orpheus_clip_composer_app Build optimized Release version of Clip Composer
Clean build rm -rf build && cmake -S . -B build && cmake --build build Full clean rebuild (use when build is corrupted)
Clean build Clip Composer rm -rf build/apps/clip-composer && cmake --build build --target orpheus_clip_composer_app Clean rebuild only Clip Composer (preserves SDK build)
Testing
Run all tests ctest --test-dir build --output-on-failure Run all unit and integration tests
Run tests (verbose) ctest --test-dir build --output-on-failure --verbose Run tests with detailed output
Run specific test ctest --test-dir build -R TestName --output-on-failure Run only tests matching TestName pattern
Run Clip Composer tests ctest --test-dir build -R ClipComposer --output-on-failure Run only Clip Composer-specific tests
Run OCC integration tests ctest --test-dir build -R OCC_Integration --output-on-failure Run OCC integration tests with SDK
Run Applications
Run Clip Composer open build/apps/clip-composer/orpheus_clip_composer_app_artefacts/Debug/OrpheusClipComposer.app Launch Clip Composer (macOS Debug build)
Run Clip Composer (Release) open build-release/apps/clip-composer/orpheus_clip_composer_app_artefacts/Release/OrpheusClipComposer.app Launch Clip Composer (macOS Release build)
Relaunch Clip Composer (clean) ./scripts/relaunch-occ.sh Clear cache, rebuild, and relaunch Clip Composer (recommended development workflow)
Start OCC instance cd ~/dev/orpheus-sdk/apps/clip-composer && claude-code Start isolated Claude Code instance for OCC development
Clip Composer (OCC) Workflow
Full OCC rebuild & launch ./scripts/relaunch-occ.sh Recommended workflow: clean, rebuild, launch OCC
Quick OCC rebuild cmake --build build --target orpheus_clip_composer_app && open build/apps/clip-composer/orpheus_clip_composer_app_artefacts/Debug/OrpheusClipComposer.app Fast rebuild and launch (no cache clear)
View OCC logs tail -f ~/Library/Logs/OrpheusClipComposer/*.log Monitor OCC application logs in real-time
Clear OCC preferences rm -rf ~/Library/Application\ Support/OrpheusClipComposer/ Reset OCC to default settings (removes sessions)
OCC development instance cd ~/dev/orpheus-sdk/apps/clip-composer && claude-code Start isolated Claude Code instance for OCC work
View OCC docs index cat apps/clip-composer/docs/occ/INDEX.md Browse complete OCC documentation catalog
Check OCC version grep -A 3 "PROJECT_VERSION" apps/clip-composer/CMakeLists.txt Show current OCC version from CMake config
OCC v0.2.0 QA checklist cat apps/clip-composer/docs/occ/OCC104*.md View v0.2.0 sprint plan and QA checklist
Git Operations
Check status git status View current branch and changed files
View diff git diff See unstaged changes
View recent commits git log --oneline -10 Show last 10 commits
Fetch updates git fetch origin Download latest changes from remote (doesn't merge)
Pull latest git pull origin main Fetch and merge latest changes from main branch
Code Quality
Format all code find src apps -name '*.cpp' -o -name '*.h' | xargs clang-format -i Auto-format all C++ source files
Check formatting find src apps -name '*.cpp' -o -name '*.h' | xargs clang-format --dry-run -Werror Verify code formatting without modifying files
Documentation

View OCC progress cat apps/clip-composer/.claude/implementation_progress.md View Clip Composer implementation progress

Diagnostics
Check CMake version cmake --version Verify CMake installation
Check build tree tree build -L 3 -d View build directory structure
List build targets cmake --build build --target help Show all available build targets
Check OCC build artifacts ls -lh build/apps/clip-composer/orpheus_clip_composer_app_artefacts/Debug/ List OCC Debug build artifacts and app bundle
Verify OCC app bundle file build/apps/clip-composer/orpheus_clip_composer_app_artefacts/Debug/OrpheusClipComposer.app/Contents/MacOS/OrpheusClipComposer Check OCC executable type and architecture
Check OCC dependencies otool -L build/apps/clip-composer/orpheus_clip_composer_app_artefacts/Debug/OrpheusClipComposer.app/Contents/MacOS/OrpheusClipComposer List dynamic library dependencies for OCC (macOS)
View OCC build log cmake --build build --target orpheus_clip_composer_app 2>&1 | tee occ-build.log Build OCC and save output to log file
